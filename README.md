# pixelebbe-bot
low effort SIP client bot playing [pixelebbe](https://pixeleb.be/), created during GPN24
built using opensip (very alpha barely functional pure python sip client), using whisper and basic silence for detection.   
web ui, live monitoring, stats and queue system that watches and queues pixels that need to be changed.  
speed depends heavily on availability, typically ~140 pixels per hour

full of bugs  
any credentials you find aren't valid
